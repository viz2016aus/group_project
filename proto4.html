Prototyp 4

Idee

Bei diesem Prototypen kann sich der Benutzer den Startpunkt völlig frei aussuchen. Die Anzahl der Ausgewählten Felder sollte zu keinem Übersichtsverlust führen. Farben und Größe kann frei eingestellt werden und die Übersicht weiter verbessern. Funktionalität kann über eine Legende oder über Tooltipps erfolgen.


Ablauf

  1. Diese Anzeige bleibt immer gleich. Bei Tagen stehen nur die zur Auswahl die tatsächlich in den Daten enthalten sind. Start gibt an, bei welcher Messstation begonnen werden soll

  2. In diesem Bereich werden die schon ausgewählten Elemente angezeigt. Durch klicken auf + oder - kann man Elemente hinzufügen oder entfernen. Jahr, Monat und Tage sind allgemein gültig. Bei Start stehen die jeweiligen Messstationen zur auswahl die sich auf dieser Autobahn (gleiche Zeile) befinden.Mit einem Klick auf die Farbe kann zwischen mehreren verschiedenen Farben wählen.

  3. Das + und - an dieser Stelle sind nur aktiv wenn schon eine weitere Autobahn ausgewählt wurde.

  4. Hier kann bestimmt werden welche Daten berücksichtigt werden.

  5. An dieser Stelle kann man weitere Messpunkte hinzufügen oder entfernen. Der letzte Pfeil (>>) fügt alle Punkte bis zum ende der Autobahn hinzu.

  6. Dieser Strahl stellt die Autobahn dar. Es wird jeweils nur der Start und der Endknoten mit Namen angezeigt. Die Messstationen die die dazwischen liegen werden nur durch kleine Kreise dargestellt. Wird eine zweite Autobahn ausgewählt kommt ein zweiter Strahl direkt darunter und stellt die zweite Autobahn dar.

  7. Mit den Pfeilen kann die gewünschte Richtung eingegeben werden. Mit einem Doppelklick auf Richtung werden alle Daten kompiniert

  8. In dieses Fenster kommen die Eigentlichen "Ergebnisse" der Auswahl. Die Messpunkte werden in gewünschter Richtung aufaddiert und in der ausgesuchten Farbe dargestellt. Ist eine Zweite Autobahn ausgewählt kommt ein zweiter Balken direkt dazu. Auf Wunsch kann ein Raster zugeschalten werden. Der Schwarz umrandete Bereich soll als jpg exportiert werden können.
